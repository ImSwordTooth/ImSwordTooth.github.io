设计图裁剪工具 - split

设计图裁剪工具 - split ​

精心制作的玩具项目。 github 地址。

以设计图为背景，在上层手动绘制出矩形，矩形可嵌套。矩形就是组件的抽象，矩形的嵌套关系就是组件的嵌套关系。
每个组件还可以设置propTypes、预引入的组件、埋点信息、碎片等。
设置完毕后可生成为本地文件，直接初始化一个具备完整结构的React或Vue项目。

技术栈 ​

类型 说明
框架 React（Class写法，为了兼容 pixi.js）
CSS styled-components
构建工具 webpack
重要依赖 pixi.js

画布部分 ​

画布是本项目最重要核心的部分，由 2D WebGL 渲染引擎pixi.js完成。
虽说是用了第三方库，但是图形绘制和操作的逻辑也没有很少。

绘制矩形 ​

绘制矩形时要考虑以下情况：

  • 从上往下从左往右只是一般情况，很有可能是反向绘制
  • 鼠标按下时进行hittest，如果没有命中，就在最外层容器绘制，如果命中，就在命中的矩形里绘制，形成嵌套关系
  • 每次绘制时重新生成一个随机色，如果是嵌套的矩形，就使用父矩形的系列颜色的随机色（使用了randomcolor）
  • 画布可能被手动移动、缩放等
  • 如果只移动了很短的距离，不会创建

绘制与操作顶点 ​

选中矩形时，根据矩形顶点绘制出四个圆形。
每个圆形都添加mousedown、mouseover、mouseup事件以控制矩形的尺寸。
mouseover时实时修改矩形尺寸，x 和 y 值最小的为矩形左上角，取其对角点，可获取矩形宽度和高度。
