多彩标签 - NameTag ​

适用于人物姓名的标签，每一个人都分配一种颜色，鼠标 hover 时，同一人所在的标签都会高亮，强调是同一人。

比如以下场景：

标题 内容 发布时间 发布人
宣州谢朓楼饯别校书叔云 弃我去者，昨日之日不可留； 乱我心者，今日之日多烦忧。 2023.1.17
李白
归去来兮辞 悟已往之不谏，知来者之可追注。 2023.1.17
陶渊明
滕王阁序 豫章故郡，洪都新府。星分翼轸，地接衡庐。 2023.1.17
王勃
蜀道难 飞湍瀑流争喧豗，砯崖转石万壑雷。 2023.1.17
李白
原理

每当有一个新的人物出现，都会更新 localStorage 里的人物列表。
更新时，会通过 RandomColor 随机生成一个暗色，和人物名称一起 push 到列表里。
如果人物在列表里有，就获取储存的色值，并放到 dom 的 css 里。

相同人物的 class 也一样，class 的值为 people-${children}，因此，hover 的时候，获取有这个 class 的 dom 列表，循环修改 css 即可。

原理不变，样式上就可以随意了，每个 NameTag 的样式可以自己随意定制。

vue
<script setup>
	onMounted(() => {
		const arr = JSON.parse(localStorage.getItem('nameArr') || '[]') || [];

		if (!arr.find(a => a.name === childrenSlot.value)) {
			const color = RandomColor({luminosity: 'dark'})
			const newList = [
				...arr,
				{
					name: childrenSlot.value,
					color
				}
			]
			localStorage.setItem('nameArr', JSON.stringify(newList));
			peopleColor.value = color
		} else {
			peopleColor.value = arr.find(a => a.name === childrenSlot.value)?.color
		}
	})
</script>
