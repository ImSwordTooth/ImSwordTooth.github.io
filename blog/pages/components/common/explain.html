# 注解 - Explain

overscroll-behavior 是CSS的新属性，允许开发者覆盖默认的浏览器滚动行为，有三个值：

  • auto：其认值。元素（容器）的滚动会传播给其祖先元素。有点类似 JavaScript 中的冒泡行为。
  • contain：阻止
    滚动链接
    。 滚动行为不会传播给其祖先元素，但会影响节点内的局部显示。例如，Android 上的光辉效果或 iOS 上的回弹效果。当用户触摸滚动边界时会通知用户。注意，overscroll-behavior:contain 在 html 元素上使用，可以阻止导航滚动操作。
  • none：和 contain 一样，但它也可以防止节点本身的滚动效果。 也可以设置 overscroll-behavior-x 和 overscroll-behavior-y 来分别设置两个方向上的滚动行为。
支持大部分
markdown 语法
。
亮点
  • 不占据文档的布局，阅读注释时也不会打扰正式内容的进度
  • 基本支持所有 markdown 文件，可以手写，也可以require一个现成的.md文件
原理
  1. 给文本添加一个mouseover事件
  2. 触发后，使用showdown库把传入的md转成html字符串
  3. 把html字符串放到body元素上，但是隐藏不显示，获取其高度后移除
  4. 把html字符串和高度放到data里，以对应的 vue 组件的 v-html 和 :style
