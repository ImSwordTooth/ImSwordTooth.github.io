# 立即编辑文本 - LabelInput

我是标题，点击修改我

我是龟缩在一个角落的小文本，但是支持修改

还可以只读，虽然没啥用。。
亮点
  • 立即在原地编辑，节省空间
  • 带有小图标，提示用户可编辑
  • 点击空白处或者按Enter时会触发更新，方便
  • 可定制文本和文本框的样式
原理
  • 主要功能就是一个 if 语句，切换 input 和 span，没什么可说的
  • 点击空白处的原理是，当 input 显示出来时，document添加一个点击事件，如果该事件的event.target不在input内，就关闭 input，并移除该事件。
<!--基础用法-->
<LabelInput value="我是标题，点击修改我" @Change=""></LabelInput>

<!--可自定义样式-->
<LabelInput
    value="支持修改"
    :style="{ margin: '0 4px' }"
    :inputStyle="{ fontSize: '12px', width: '120px', padding: '2px 4px' }"
    @Change="">
</LabelInput>

interface LabelInput {
    value: string; // 文本
    onChange: Function // 更新函数
    style?: Object; // 文本样式
    inputStyle?: Object // 文本框样式
    size?: String // antd 的文本框的 size
    readOnly?: Boolean // 为 true 时只能查看，不能编辑
}
