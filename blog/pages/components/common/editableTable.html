# 立即编辑表格 - EditableTable

点击首列的-来删除该行。
在最后一行输入数据来添加一行。

亮点
  • 编辑数据简单高效
  • 带有删除数据和添加数据
  • 所有操作都不会占据额外的空间
原理

落实到代码上都是挺一般操作，没什么好说的。

<EditableTable
	:dataList="[{a:'11', b: '22'}, { a:'33', b: '44'}]"
	:columnList="[{title:'hello', value: 'a'}, { title:'world', value: 'b'}]"
	:update="changeData" />

interface Explain {
	dataList: Array; // 数据列表
	columnList: Array<{  // 表头列表
		title: string // 显示出来的表头文本
		value: any // 对应的数据字段
		width?: string // 格子宽度，可以不写（默认auto）、px、%
	}>;
	update: () => {}; // 释义文本
}
