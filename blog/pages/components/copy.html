复制 - Copy ​

点我复制 ^_^

带有复制小图标，提示用户此处可以复制，而且混迹在其他任何地方

都不会显得突兀。

可以传入 svg
或者 img
作为 slot。

或者自定义颜色

。

默认弹出复制成功提示，也可以传入

自定义函数
。
javascript
const tag = document.createElement('input');
tag.setAttribute('id', 'CopyComponentTempInput');
tag.value = slots.default()[0].children;
document.body.appendChild(tag);
tag.select();
document.execCommand('copy');
document.body.removeChild(tag)
