# 复制 - Copy

点我 ^_^
亮点
  • 带有复制小图标，提示用户此处可以复制，而且混迹在其他任何地方都不会显得突兀
  • 只需要点击即可完成复制
  • 复制之后提示复制成功
原理
  1. 使用原生的 createElement() 创建一个隐藏的 input 元素，把要复制的值放进去。
  2. 然后再使用 document.execCommand('copy') 把值复制到剪切板。
  3. 最后删掉这个隐藏的 input。
<Copy>点我 ^_^</Copy>

interface Copy {
    children: string; // 要复制的文本
}
