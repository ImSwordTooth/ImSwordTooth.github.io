# 单容器 - Container

布局组件

普通的容器组件，可以认作是一个普通的div。可以在里面放若干子元素。
本组件有个是否居中的布尔值 props: isCenter，实现原理就是margin: 0 auto，可能会和组件本身的 style 有冲突。待优化。
搭配宽度的设置，可实现活动页常见的主宽度。

# props

export interface ContainerProps {
    style?: object // 组件样式
    width: number // 宽度
    isCenter: boolean // 是否居中
    children?: Element // 子元素
}
