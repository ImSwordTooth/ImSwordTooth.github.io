# 布局容器 - Grid

布局组件

类似于栅格，将一行分割成若干个区域。
默认按比例分割，最大份数是12，可分为：

  • 12
  • 6:6
  • 3:9
  • 9:3
  • 4:4:4
  • 3:6:3

选择比例之后，还可以将某个比例换成特定的像素值，比如 3:400px、4:4:120px，基于flex布局完成。

# props

interface GridProps {
    gridList: Array<number|string> // 比例, 如果是带px的，就说明是个具体值
    gap: number // 间距
}
